Beginner-friendly

  • PICO-8: A "fantasy virtual console", which is actually a 128x128 and 16 colors mini-engine. It uses Lua as a programming language, and features a very small set of instruction.
  • FREE Bitsy: A simple tool to make retro, short narrative games or tiny adventures.
  • FREE Twine: An engine for text-based narrative games.

For people who (would) like to code

  • free Defold: 2D game engine aimed for mobile and Web games, by King. It uses Lua.
  • free Phaser: JavaScript 2D game framework, for Web games.
  • Unity: 3D and 2D game engine, can build for most platforms. It has a free tier. Coding is done with C#.
  • GameMaker: 2D game engine, can build for most platforms. Coding is done with a custom scripting language.

Books

  • A Theory of Fun for Game Design by Raph Koster: a must-read book that explains why humans find games fun.
  • Game Design Workshop by Tracy Fullerton: a very hands-on approach to game design, prototyping, project iteration, play testing, etc.

üñå Art

Articles and tutorials

  • The Pixel Art Tutorial: a good introduction to making pixel art. It covers the main techniques, such as drawing lines, anti-aliasing, dithering, choosing a palette, etc.

  • Pedro Medeiro's pixel art tutorials in Patreon: These are short, visual lessons in GIF format, each one explaining a specific point, by the artist of games such as Celeste or Towerfall.

Assets

  • Kenney Assets: high quality art assets for games, with a Public Domain license.
  • OpenGameArt: collection of assets for games (mostly images) with Creative Commons licenses.

Tools & apps

  • Pyxel Edit Pixel art editor (Win and Mac).
  • Aseprite Another popular pixel art editor (Win, Linux and Mac)
